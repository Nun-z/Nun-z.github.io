# Nun-z.github.io

This method (used first by tunip3) allows apps to be installed to an Xbox in retail mode that would normally only be available in dev mode.

RetroArch and PPSSPP are available and functional, with the others being WIP.

I'm working on a guide to explain how to install these sorts of apps to an Xbox in retail mode.  In order to avoid breaking Microsoft's terms of service, the guide will use an FTP app for the tutorial, but any app can be added to retail mode in this way.
